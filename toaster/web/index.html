Help — Toaster

Label lidar point clouds in 3D: navigate, select points, assign semantic classes. Run a segmenter so one click labels a whole cluster.

Select

  • Point mode — click a point to select; double-click to label it.
  • Box mode — left-drag a rubber band to select many; double-click inside to label.
  • Voxel mode — a click selects the whole voxel.
  • Shift adds to the selection · Ctrl removes from it.

Label

  • Enter assign the active class to the selection
  • 0–9 pick that class and assign in one go
  • Esc clear the selection
  • Ctrl+Z / Ctrl+Y undo / redo

Camera

  • Point / Voxel — left-drag orbit · right-drag pan · wheel zoom
  • Box — right-drag orbit · Shift+right-drag (or middle) pan · wheel zoom
  • ←→ roll · ↑↓ pitch · Shift+←→ yaw
  • WASD fly — forward / left / back / right (physical keys — ZQSD on AZERTY) · hold Shift to go faster

Frames — apairo dataset

  • , / . previous / next frame
  • Dataset window — switch sequence & channel; click the counter to jump to a frame

Segment

  • Pick a segmenter, set its params, Run — clusters appear in the Segments window.
  • Tick segments and Assign checked, or double-click a cluster to label it.
  • Ground filters — tick Use camera view as up after turning the scene upright.

Save

  • Save writes labels + a class-names YAML beside the cloud (or a folder you pick / create).
  • In an apairo dataset — write the labels back as a channel (ground_truth).

Display

  • Views — labels · grouping · intensity · height. Point size & round points in the Display window.
